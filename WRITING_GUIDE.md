## Book Format (Long-form Narratives)

Create files in `_books/` for chapters:

```markdown
---
layout: book
title: "Chapter Title"
book: "Book Name"
chapter: 1
---

Chapter content here...
```

## Tips for Writing

- Write in whatever style feels natural
- Don't worry about perfect grammar - I can help edit later
- Add details that make the memory vivid
- Include emotions and sensory details
- Date and time context helps place memories
